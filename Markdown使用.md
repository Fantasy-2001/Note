## 调到本页指定位置

在我们使用Markdown写文章的时候,有时候文章很长,就需要索引,有些人不需要看前面的部分,或者想直接看某条索引下的文字,这种页面内跳转,就需要本文介绍的---锚点!
 这里主要使用了HTML的一个标签: < a> < /a>
 总共需要下面三步:

```markdown
1. 添加链接：通过添加标签<a>标题一</a>
2. 添加锚点：在需要跳转到的标题位置添加链接
<a name="锚点名称">标题一</a>
3. 修改1中的标签<a hrer="#要跳转到的锚点名称">标题一</a>

例子：
<a href="#1">第一段</a>
<a href="#2">第二段</a>
<a href="#3">第三段</a>

<a name="1">第一段</a>
...
<a name="2">第二段</a>
...
<a name="3">第三段</a>
```
