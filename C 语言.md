## 格式化输出/输入

**转换说明及其打印的输出结果**

| 转换说明 |                             输出                             |
| :------: | :----------------------------------------------------------: |
|    %a    |           浮点数、十六进制数和 p 记书法（C99/C11）           |
|    %A    |           浮点数、十六进制数和 p 记数法（C99/C11）           |
|    %c    |                           单个字符                           |
|    %d    |                       有符号十进制整数                       |
|    %e    |                       浮点数，e 记数法                       |
|    %E    |                       浮点数，e 记数法                       |
|    %f    |                     浮点数，十进制记数法                     |
|    %g    | 根据值的不同，自动选择 %f 或 %e。%e 格式用于指数小于 -4 或者大于或等于精度时 |
|    %G    | 根据值的不同，自动选择 %f 或 %E。%E 格式用于指数小于 -4 或者大于或等于精度时 |
|    %i    |                 有符号十进制整数（与%d相同）                 |
|    %o    |                       无符号八进制整数                       |
|    %p    |                             指针                             |
|    %s    |                            字符串                            |
|    %u    |                       无符号十进制整数                       |
|    %x    |            无符号十六进制整数，使用十六进制数 0f             |
|    %X    |            无符号十六进制整数，使用十六进制数 0F             |
|    %%    |                        打印一个百分号                        |
